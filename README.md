# JavaScript From Scratch

My personal journey learning JavaScript from the ground up — every concept, with notes and runnable code examples.

## Goal

Build a solid understanding of JavaScript fundamentals through to advanced topics, one concept at a time, with my own notes and practice code for each.

## Structure

| Folder | Topics Covered |
|---|---|
| `01-basics` | Variables, data types, operators |
| `02-control-flow` | Conditionals, loops |
| `03-functions` | Function declarations, arrow functions, scope, closures |
| `04-arrays-objects` | Arrays, objects, array/object methods |
| `05-dom` | DOM manipulation, events |
| `06-advanced` | Async JS, promises, ES6+ features, classes |

## How to Run

Each `.js` file can be run with Node:

\`\`\`bash
node 01-basics/01-variables.js
\`\`\`

Or pasted directly into a browser console.
